# Life Planner Web App

A simple Life Planner Web App to organize your daily tasks, goals, and personal productivity.

## Features
- Daily task planning
- Goal tracking
- Clean and simple interface
- Easy to use for students and professionals

## How to Use
1. Download or clone this repository.
2. Open the file `index.html` in your browser.
3. Start planning your day!

## Technologies Used
- HTML
- CSS
- JavaScript

## Purpose
This project is designed to help users stay organized and manage their daily life efficiently.
